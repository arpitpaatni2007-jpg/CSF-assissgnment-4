This repository contains the source code and documentation for Assignment 4, completed as part of the Computer Science Fundamentals and Career Pathways course.

The project is a fully responsive portfolio website created using HTML and CSS, following all requirements of the assignment. It includes structured sections such as About, Skills, Projects, and Contact, along with a clean and proffesional theme

Features Include:

Elegant and responsive portfolio layout
Professional and modern theme
Clean navigation bar
Introduction hero section
About Me section with personal details
Skills represented using responsive styled badges
Project cards showing completed work
Contact form UI
Accessible layout with readable typography
